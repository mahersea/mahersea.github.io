# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Build/Run Commands
- Start server: `npm start` or `node server.js`
- Open project: Open `index.html` in a browser

## Code Style Guidelines
- HTML: Clean semantic markup with proper indentation
- CSS: Use consistent class naming with descriptive names
- JavaScript:
  - ES6+ syntax preferred
  - Descriptive variable/function names (camelCase)
  - Use comments for complex logic
  - Organize code with classes for encapsulation
  - Favor const/let over var

## Project Structure
- HTML files have inline CSS and JavaScript
- External data in JSON format (airports.json)
- Use p5.js for canvas rendering and animations
- Use p5.sound for audio functionality

## Dependencies
- Bootstrap for basic styling
- p5.js for canvas visualization
- p5.sound for audio synthesis
