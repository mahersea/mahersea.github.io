# GeoJSON Map Visualization

An interactive geographic data visualization using GeoJSON and Leaflet.js.

## About
This project demonstrates how to visualize geographic data using the GeoJSON format and the Leaflet.js mapping library. It provides an interactive map interface that can display various types of geographic features (points, lines, polygons) with customizable styling and popup information.

## Features
- Interactive map with pan and zoom capabilities
- GeoJSON data visualization with customizable styling
- Layer control for toggling different data sets
- Popup information on feature click
- Responsive design for different screen sizes

## GeoJSON Overview
GeoJSON is a format for encoding geographic data structures using JavaScript Object Notation (JSON). The visualization supports various GeoJSON feature types:
- Points (markers) for locations
- LineStrings for paths and routes
- Polygons for areas and boundaries
- Feature properties for additional data
