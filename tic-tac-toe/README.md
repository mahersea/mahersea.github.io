# Tic-Tac-Toe with Minimax AI

A classic Tic-Tac-Toe game with an unbeatable AI opponent using the minimax algorithm.

## About
This implementation of Tic-Tac-Toe features a computer opponent that uses the minimax algorithm to make optimal decisions. The AI analyzes all possible future moves to determine the best play, making it effectively unbeatable.

## Features
- Clean, minimalist interface with canvas rendering
- Human player (X) vs. AI opponent (O)
- Unbeatable AI using the minimax algorithm
- Game state tracking (win, lose, or draw)
- One-click game reset

## How to Play
Click on any empty cell to place your X. The computer will respond with an O. Try to get three in a row horizontally, vertically, or diagonally to win!
