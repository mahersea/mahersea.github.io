# Color Theory Explorer

An interactive visualization tool for learning about color theory and experimenting with different color harmonies on a dynamic color wheel.

## Description

This application provides an intuitive interface for exploring color relationships through a color wheel visualization. Users can select different color schemes (complementary, analogous, triadic, or tetradic) and interactively rotate the palette to see how colors work together in various configurations.

## Features

- Interactive color wheel visualization
- Multiple color harmony schemes (complementary, analogous, triadic, tetradic)
- Dynamic palette generation
- Drag-to-rotate palette functionality
- Light/dark mode toggle
- Educational descriptions of each color scheme
- Real-time palette preview

## Technical Details

The implementation utilizes:
1. Canvas-based color wheel rendering
2. HSL to RGB color conversion
3. Interactive marker dragging with angle calculation
4. Dynamic UI updates based on user interactions
5. CSS variables for theming support
