# Drone Bomber Game

A canvas-based scrolling game with a top-down view of a battlefield where players control a drone.

## Description

This project creates a vertically scrolling game environment with various terrain features and moving vehicles. The current implementation focuses on the scrolling background mechanism and object movement, laying the groundwork for a complete game where players would control drones to target enemy vehicles.

## Features

- Smooth scrolling terrain background
- Dynamic moving vehicles with autonomous pathfinding
- Variety of terrain elements (trees, roads, rivers, craters)
- Double-buffered rendering using offscreen canvas
- Physics-based object movement

## Technical Details

The game utilizes:
1. Canvas API for all rendering
2. Double-buffering for efficient background scrolling
3. Procedurally generated terrain elements
4. Independent object movement with collision detection
5. Animation frame-based game loop
