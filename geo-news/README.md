# News Events Map

An interactive geographic visualization of news events with filtering capabilities.

## Description

This application displays news events on a map interface, allowing users to explore local news based on geographic location. The map shows event markers that can be filtered by category and time range, providing a spatial context for news stories. When a user clicks on an event marker, a popup displays details about the news story with a link to the full article.

## Features

- Canvas-based map visualization
- Interactive markers for news events
- Category-based filtering system
- Time range selection (24 hours, week, month)
- Event popup with story details
- Zoom and pan map controls
- Reset view functionality
- Loading indicator for data fetching
- Responsive layout for different screen sizes

## Technical Details

The implementation utilizes:
1. HTML5 Canvas for map rendering
2. Custom map implementation with JavaScript
3. Event handling for marker interaction
4. Dynamic category filter generation
5. Time-based data filtering
6. Popup information display system
7. External JavaScript modules for map and event handling
