# Two.js Interactive Explorer

An interactive playground for experimenting with the Two.js vector graphics library, allowing you to create and manipulate shapes in real-time.

## Description

This application provides an intuitive interface for exploring the Two.js library's capabilities. The left panel offers a code editor with predefined examples and API documentation, while the right panel shows a live preview of the rendered graphics. This sandbox environment makes it easy to learn and experiment with vector graphics programming.

## Features

- Interactive code editor with instant preview
- Predefined example snippets for common Two.js operations
- Basic API reference integrated into the interface
- Console logging for debugging and learning
- Support for creating circles, rectangles, ellipses, polygons, and stars
- Animation capabilities with requestAnimationFrame
- Shape grouping and transformation functionality

## Technical Details

The implementation utilizes:
1. Two.js for vector graphics rendering
2. Event-driven code execution
3. Dynamic DOM manipulation
4. Eval-based code interpretation for the playground
5. Interactive help system with example loading
