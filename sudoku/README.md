# Sudoku Game with Solver

An interactive Sudoku game with automatic solving capability using backtracking algorithm.

## About
This Sudoku implementation allows players to solve puzzles manually or use an automatic solver. The solver uses a backtracking algorithm to find the solution efficiently by trying numbers and undoing incorrect attempts.

## Features
- Clean, intuitive interface with a 9x9 grid
- Manual solving with keyboard number input
- Different visual styling for original clues vs. player inputs
- Automatic puzzle solver using backtracking algorithm
- Solution validation to check for correct puzzle completion
- Reset button to start over

## How to Play
Click on any empty cell and type a number from 1-9. Use Backspace or Delete to clear a cell. The solver button will automatically complete the puzzle for you.
