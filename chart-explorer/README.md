# Chart.js Explorer

An interactive playground for creating and customizing charts using the Chart.js library.

## Description

This application provides a user-friendly interface for experimenting with Chart.js visualizations. The left panel offers controls for chart type selection and data configuration, while the right panel displays the rendered chart. This tool makes it easy to visualize data and understand how different chart types represent the same information.

## Features

- Support for multiple chart types (bar, line, pie)
- Interactive data input with comma-separated values
- Custom label definition capabilities
- Real-time chart updates with a single click
- Responsive design with flexible layout
- Automatic chart re-rendering on configuration changes
- Pre-defined color schemes for data visualization

## Technical Details

The implementation utilizes:
1. Chart.js library for data visualization
2. Dynamic chart configuration based on user input
3. Instance management to handle chart updates
4. Flexible layout using CSS Flexbox
5. Form controls for intuitive data entry
