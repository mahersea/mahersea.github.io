# GraphQL - Exploring Countries

An interactive single-page application for exploring GraphQL queries against the public Countries API.

## Description

This application provides a user-friendly interface for experimenting with GraphQL queries using the Countries API. The playground allows users to write and execute GraphQL queries, view the JSON responses, and explore country, continent, and language data through a set of predefined example queries. The interface includes comprehensive documentation and quick links to help users understand GraphQL query syntax.

## Features

- Interactive GraphQL query editor with syntax highlighting
- Live query execution against a real-world Countries API
- Quick links for common query examples (countries, continents, languages)
- Dark/light theme toggle for comfortable viewing
- Detailed documentation with example queries
- Optional canvas background toggle
- Responsive design that works across devices
- JSON response formatting for readability

## Technical Details

The implementation utilizes:
1. Vanilla JavaScript for all interactions
2. Fetch API for GraphQL queries
3. CSS variables for theming
4. Single-page application architecture
5. HTML5 Canvas for optional visual elements
6. GraphQL POST requests for data retrieval
7. JSON pretty-printing for response formatting
