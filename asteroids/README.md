# Asteroids Game

A modern implementation of the classic arcade game Asteroids with smooth controls and visual effects.

## About
In this game, you control a spaceship navigating through an asteroid field. Your goal is to destroy asteroids by firing torpedoes while avoiding collisions. The game features momentum-based movement physics and particle effects.

## Features
- Classic arcade gameplay with smooth controls
- Physics-based movement with momentum and inertia
- Hyperspace jump for emergency escapes
- Particle explosion effects
- Dark/light mode toggle
- Score and lives tracking

## Controls
- Arrow keys: Rotate ship
- Space: Thrust
- T: Fire torpedo
- H: Hyperspace jump
