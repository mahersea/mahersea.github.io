# LittleTaskMan

A lightweight task management application with user, project, and Gantt chart visualization capabilities.

## Description

LittleTaskMan is a comprehensive yet simple task management system designed to help teams organize and track work. The application provides a clean interface for managing tasks, users, and projects, with an integrated Gantt chart for visualizing project timelines. The system is built using modern web technologies and follows a responsive design approach.

## Features

- Task management with status tracking (Open, In Progress, Completed, Blocked, Postponed)
- User assignment and management
- Project organization and categorization
- Interactive Gantt chart visualization
- Clean, tab-based navigation interface
- Form-based data entry with validation
- Responsive design for various screen sizes
- Error handling system
- Custom branding with logo integration

## Technical Details

The implementation utilizes:
1. Server-side API for data persistence with Node.js
2. Client-side JavaScript for interactive elements
3. CSS for responsive layout and styling
4. Tabbed interface for content organization
5. Form validation for data integrity
6. JSON-based data storage
7. Custom error handling system
8. Dynamic table generation for data display
