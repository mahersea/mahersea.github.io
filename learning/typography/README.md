# HTML Special Characters Reference

An interactive index and viewer for HTML special characters, entities, and mathematical symbols.

## Description

This application serves as a comprehensive reference for HTML special characters and mathematical symbols. It features a two-panel interface with an index of characters on the left and detailed information with a preview on the right. The tool is designed to help web developers and content creators quickly find and understand how to use special characters in their HTML documents.

## Features

- Comprehensive list of HTML entities and special characters
- Interactive preview with large display of selected characters
- Display of entity names, numeric codes, and actual symbols
- Dark/light mode toggle for different viewing preferences
- Responsive layout with scrollable index
- Information on HTML escaping sequences for special characters
- Mathematical symbols and typographic marks included

## Technical Details

The implementation utilizes:
1. CSS variables for theme switching
2. Dynamic content generation from a JavaScript data structure
3. Event-driven UI updates
4. Clean separation of content and presentation
5. Semantic HTML structure with appropriate code formatting
6. Cross-browser compatible character entity handling
