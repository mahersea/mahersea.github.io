# Flexbox Playground

An interactive tool for learning and experimenting with CSS Flexbox layout properties.

## Description

This application provides a hands-on environment for exploring CSS Flexbox layout capabilities. Users can modify various Flexbox properties through dropdown menus and see the changes reflected in real-time. The tool includes a live code preview showing the generated CSS, making it easy to understand how different Flexbox settings affect layout behavior.

## Features

- Interactive controls for Flexbox properties (direction, justify-content, align-items, wrap)
- Live preview of layout changes
- Real-time CSS code display
- Add/remove flex items dynamically
- Light/dark theme toggle
- Optional canvas background toggle
- Responsive design for various screen sizes

## Technical Details

The implementation utilizes:
1. CSS variables for theming and dynamic styling
2. DOM manipulation for interactive controls
3. Event listeners for real-time updates
4. HTML5 Canvas for background effects
5. CSS Flexbox for both the demo container and overall layout
