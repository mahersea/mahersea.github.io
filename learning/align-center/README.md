# Intro to CSS Centering Methods

An interactive tool for learning and comparing different CSS centering techniques.

## Description

This application provides a hands-on environment for exploring various HTML/CSS centering methods. Users can select different centering techniques from a dropdown menu and see both the code and the visual result in real-time. The playground is designed to help developers understand the nuances of different centering approaches and their appropriate use cases.

## Features

- Interactive code editor with live preview
- Multiple centering techniques (text-align, margin auto, flexbox, grid, absolute positioning)
- Copy-to-clipboard functionality for easy code reuse
- Light/dark theme toggle with persistent settings
- Responsive design with split-panel layout
- Real-time code application

## Technical Details

The implementation utilizes:
1. CSS variables for theming and color management
2. DOM manipulation for dynamic content updates
3. LocalStorage for theme preference persistence
4. Pre-defined code snippets demonstrating each technique
5. Clipboard API for code copying functionality
