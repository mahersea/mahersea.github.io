# Internet Protocols Learning Console

An interactive learning resource for exploring internet protocols, HTTP/HTTPS concepts, and API testing tools with a clean, accessible interface.

## Description

The Internet Protocols Learning Console is a comprehensive educational tool designed to help users understand key networking protocols and how they enable data exchange across the internet. This single-page application provides in-depth information on HTTP/HTTPS, FTP/SFTP, WebSockets, and other communication protocols, along with detailed guides for using tools like Postman and cURL for API testing.

The console features a clean, organized interface with an intuitive navigation system, allowing users to easily browse through various topics from basic protocol concepts to advanced authentication techniques. All content is presented in a readable format with clearly structured sections and subsections for optimal learning.

## Features

- Comprehensive coverage of internet protocols (HTTP, FTP, WebSockets, etc.)
- Detailed explanations of HTTP methods, request crafting, and response handling
- Practical examples for using Postman and cURL for API testing
- Advanced topics including authentication, error handling, and automation
- Dark/light mode toggle for comfortable reading in any environment
- Responsive design that works across desktop and mobile devices
- Clean, intuitive navigation with clearly organized menu structure
- Detailed code examples for practical implementation

## Technical Details

- Built with HTML, CSS, and vanilla JavaScript
- Implements custom theme switching functionality
- Uses CSS variables for easy theme management
- Responsive design with media queries for mobile compatibility
- Dynamic content loading without page refreshes
- Organized JavaScript structure with content stored as objects
- Clean modular code separation between content and functionality
- Gradient-based styling for visual appeal
