# TypeScript Basics Interactive Guide

An interactive learning tool for TypeScript fundamentals with interactive code snippets and detailed explanations.

## Description

This application provides an accessible introduction to TypeScript core concepts through interactive code examples. Users can select from a variety of TypeScript topics such as type annotations, interfaces, classes, generics, and more to explore syntax and usage patterns. Each code snippet is fully interactive, with hoverable elements that reveal detailed explanations about specific parts of the syntax.

The guide creates an educational experience that bridges theory and practice, allowing users to understand TypeScript concepts without needing to compile or run code. The clean, modern interface includes a dark/light mode toggle for comfortable viewing in different environments, and each section includes links to official TypeScript documentation for further learning.

## Features

- Dropdown menu with 12 core TypeScript topics
- Interactive code snippets with hover-activated explanations
- Detailed topic overviews with links to official documentation
- Dark/light mode toggle for comfortable viewing
- Responsive layout for various screen sizes
- Color-coded syntax highlighting
- Split-pane interface with code and explanation sections
- Seamless topic switching without page reloads
- Mouseover highlighting for specific syntax elements
- Contextual help for TypeScript-specific constructs
- Links to official TypeScript documentation for each topic
- Clean, modern UI with gradient accents

## Technical Details

- Built with vanilla JavaScript, HTML5, and CSS3
- Uses CSS variables for theming and dynamic color schemes
- Implements event-driven interaction model
- Interactive elements using data attributes
- DOM manipulation for dynamic content updates
- Custom hover effects with contextual information
- CSS transitions for smooth theme switching
- Responsive design with flexbox layout
- Modular JavaScript organization
- Structured data for code snippets and explanations
- No external dependencies or frameworks required
