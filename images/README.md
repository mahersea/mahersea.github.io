# Portfolio Image Assets

A collection of thumbnail images and other visual assets used throughout the portfolio website.

## Description

This directory contains the image assets used across the portfolio, including project thumbnails, icons, and other visual elements. It provides a centralized location for all imagery, ensuring consistent sizing and formatting.

## Features

- Project thumbnails with standardized dimensions
- Naming convention with `_thumb` suffix for thumbnail images
- Utility scripts for batch image renaming
- Consistent visual style across project previews

## Technical Details

The image collection includes:
1. Project thumbnail images (used on the main portfolio page)
2. Shell scripts for batch renaming operations
3. Source files for generated images
4. Consistent naming conventions for easy reference
