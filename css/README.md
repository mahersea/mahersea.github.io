# Portfolio CSS

Core stylesheets for the portfolio website, providing consistent styling across all pages and projects.

## Description

This directory contains the CSS files that define the visual appearance and layout of the entire portfolio site. It includes styles for typography, navigation, project cards, layout components, and responsive design elements.

## Features

- Base typography system
- Navigation and header styles
- Project card and grid layouts
- Animation and transition effects
- Responsive design breakpoints

## Technical Details

The CSS implementation includes:
1. Reset and normalization styles
2. Modular component styling
3. Flexbox and Grid layouts
4. Media queries for responsive behavior
5. Animation and transition utilities
